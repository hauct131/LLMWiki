## Definition

Mean Subtraction Normalization is a preprocessing technique used in image processing where the mean value of pixel intensities across all pixels within an image or batch of images is subtracted from each corresponding pixel intensity, effectively centering the data around zero. This helps to reduce variability caused by lighting differences and improves algorithm performance for subsequent tasks like super-resolution.

## Why It Matters

It enhances consistency in input features which can lead to more accurate learning outcomes when training deep neural networks on image datasets, particularly beneficial before applying complex models such as those used in single image super-resolution (SR).

## Variants / Related Terms

Zero Mean Normalization, Standard Deviation Scaling
