## Definition

Latent Semantic Analysis (LSA) is a technique in natural language processing that identifies patterns in the relationships between terms and concepts within large text corpora, using singular value decomposition to reduce dimensionality and uncover latent semantic structures.

## Why It Matters

LSA helps improve information retrieval and understanding by capturing underlying meanings beyond mere word co-occurrence frequencies.

## Variants / Related Terms

Singular Value Decomposition (SVD), Non-negative Matrix Factorization (NMF)
