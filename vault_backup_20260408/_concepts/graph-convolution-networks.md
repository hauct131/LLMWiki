## Definition

Graph Convolution Networks (GCNs) are neural network architectures designed for processing graph-structured data, where nodes represent entities or features, and edges denote relationships between them. They apply convolutional operations on graphs to capture local structures around each node effectively. GCNs generalize the operation of CNNs from grid data like images to irregular graph domains by learning a shared representation that respects both node attributes and topological structure.

## Why It Matters

Graph Convolution Networks enable powerful machine learning models capable of handling complex relational information, which is essential for tasks involving non-Euclidean structured datasets such as social networks or molecular structures.

## Variants / Related Terms

Chebyshev GCN (ChebConv), Weisfeiler-Lehman Networks (WLNets)
