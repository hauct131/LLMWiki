## Definition

Contrastive learning is an approach in machine learning that learns representations by teaching models to distinguish between similar (positive) pairs from different classes or dissimilar (negative) pairs within the same class. It often involves a contrastive loss function, which penalizes incorrect pairings and rewards correct ones during training.

## Why It Matters

Contrastive learning is significant for improving model discrimination capabilities in tasks like image recognition where distinguishing between fine-grained classes can be challenging.
