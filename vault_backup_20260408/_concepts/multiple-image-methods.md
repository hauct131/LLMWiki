## Definition

Multiple Image Methods refer to techniques used in image processing that leverage several blurred and/severely degraded images taken from slightly different viewpoints or under varying conditions. These methods aim to reconstruct a high-resolution (HR) version of the scene by exploiting redundancies across multiple low-quality inputs, often using complex mathematical models like inverse problems in optics.

## Why It Matters

Multiple Image Methods are crucial for enhancing image resolution beyond what is possible with a single source or viewpoint without introducing artifacts that could mislead interpretation and analysis.

## Variants / Related Terms

Phase Unwrapping, Interpolation-based Super-resolution (IBSR), Modeling of the Point Spread Function (PSF) in Multiple Images
